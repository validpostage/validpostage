Tutorial: Installing Resource Packs

  1. Press [Win]+[R] to open a ‘Run’ prompt
    Enter %appdata%\.minecraft\resourcepacks and click ‘OK’
  2. Move the desired resource packs into this .minecraft\resourcepacks\ folder (do not unzip them!)
  3. Open the game and navigate to ‘Options...’ > ‘Resource Packs...’
  4. Move the desired resource packs from ‘Available’ to ‘Selected’

Tutorial: Installing Fabric & Mods

  1. Download and run ‘Fabric Installer’
  2. On the ‘Client’ tab, select Minecraft Version 1.17.1, and click ‘Install’
  3. Press [Win]+[R] to open a ‘Run’ prompt
    Enter %appdata%\.minecraft\ and click ‘OK’
  4. In this \.minecraft\ folder, make a new subfolder titled mods
  5. Save ‘Fabric API’ to .minecraft\mods\
  6. Save any additional mods to the same folder
  7. Launch Minecraft using the ‘fabric-loader-1.17.1’ profile.
